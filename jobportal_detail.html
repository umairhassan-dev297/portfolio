Job Portal

Live Link (if available)

A comprehensive Job Portal system supporting four roles — Admin, Staff, Job Seeker, and Job Uploader — with real-time chat and Stripe payment integration.

Built a full-featured Job Portal using Laravel and MySQL that streamlines job posting, application, and hiring. The system supports four distinct roles ensuring a smooth experience for both employers and job seekers.

Managing four different user roles with separate dashboards and permissions, while also building a real-time chat system and integrating Stripe payments for premium job postings — all within a single Laravel application.

Implemented Laravel role-based access control (RBAC) for all four roles, built a chat system between job seekers and uploaders, and integrated Stripe API for secure premium transactions with email notifications via SMTP.

Key Features

  • 4 Role-based Dashboards
  • Real-time Chat System
  • Stripe Payment Integration
  • Job Listings & Applications
  • Company Registration
