Devaims Website Clone

View Live on Netlify

A multi-page company website clone built with HTML, CSS, Bootstrap, and JavaScript — including 5 separate pages, a Slick image slider, responsive layout, and a clean professional UI matching the original Devaims website.

I built this project as a frontend practice exercise to sharpen my HTML, CSS, Bootstrap, and JavaScript skills. The clone covers 5 pages of the Devaims company website — Home (index), About, Careers, Case Studies, Learning Center, and Web Development service page. Each page was built to closely match the original design with proper responsiveness, navigation, and layout consistency across all pages.

The challenge was keeping all 5 pages visually consistent — same navbar, same footer, same color scheme and spacing — while each page had its own unique sections and layouts. The Slick carousel slider on the Web Development page needed custom configuration for autoplay speed and slide count. Getting the career page's overlapping image layout and the process steps section to look right on all screen sizes also took careful CSS work.

Created a shared CSS file for common styles — fonts, colors, navbar, and footer — so all pages stayed consistent without repeating code. Bootstrap's grid system handled most of the responsive layout, with custom CSS for the unique sections on each page. The Slick slider was integrated via CDN and configured with JavaScript for autoplay, speed, and slide count per viewport. The final project was deployed on Netlify for instant access and fast load times.
