A complete office attendance management system built with Laravel — tracks daily check-ins, calculates working hours automatically, and restricts attendance marking to the office IP address only.

Built a full office attendance system at WebCrazier using Laravel and MySQL. The admin can register employees, manage their contracts, and view complete attendance history. Employees simply mark their daily attendance and the system handles all the working hours calculations automatically — no manual tracking needed. The system was deployed live and used in a real office environment.

The main challenge was making sure employees could only mark attendance from the physical office location — not from home or anywhere else. On top of that, the system needed to automatically calculate total working hours per day and per month, while also handling employee contracts and registration in a clean, role-based way.

Integrated an IP restriction API that checks the user's IP address before allowing attendance to be marked — if the IP doesn't match the office network, access is blocked. Working hours are calculated automatically using Laravel's time handling logic, comparing check-in and check-out timestamps. Role-based access control was used to cleanly separate admin and employee functionality.

Key Features

  • Employee Registration & Management
  • Daily Attendance Tracking
  • Auto Working Hours Calculation
  • IP Restriction — Office Only
  • Contract Management Module
  • Role-based Access Control
