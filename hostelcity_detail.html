HostelCity

View on Netlify

A responsive hostel booking landing page built with HTML, CSS, Bootstrap, and JavaScript — featuring room listings, a live search bar, login/signup flow, and a clean modern UI hosted on Netlify.

HostelCity is a frontend hostel booking website I built to practice and demonstrate my HTML, CSS, Bootstrap, and JavaScript skills. The site has a full hero section with a search bar, room listings, a dashboard button, and login/signup functionality. It's fully responsive across all screen sizes and was deployed on Netlify. This project shows that my skills are not just limited to backend — I can build clean, well-structured frontend interfaces too.

The main challenge was making the layout look polished and professional using only HTML, CSS, and Bootstrap — no backend, no frameworks. Getting the hero section with the full background image, search bar, and overlay text to look clean on all screen sizes required careful use of Bootstrap grid and custom CSS. The navigation with the branded logo style and the two-button login/signup layout also needed fine-tuning to feel right.

Used Bootstrap's grid system and utility classes for the responsive layout, and wrote custom CSS for the hero background overlay, typography, and button styling. JavaScript was used for the search interaction and navigation toggle on mobile. The final site was deployed on Netlify for fast load times and easy sharing — making it a quick and clean way to demonstrate frontend capability to potential employers.

Key Features

  • Hero Section with Search Bar
  • Room Listings & Explore Page
  • Login & Signup UI
  • Fully Responsive Layout
  • Dashboard Button & Navigation
  • Deployed Live on Netlify
